### 功能模块结构图

#### 后台功能需求

```txt
1：商品分类管理：商品分类实体：id、分类名称、父分类、标签（用来标记分类层级关系）、备注。

2：商品管理：商品实体：id、商品所属分类、商品名称、商品主图、商品价格、库存、销量、浏览量、评论量、详情描述、添加时间。

3：用户管理：用户实体：id、用户名、密码、邮箱、真实姓名、性别、注册时间。

4：地址管理：地址实体：id、用户、手机号、联系人、详细地址。

5：订单管理：订单实体：id、订单号、所属用户、地址、订单总价、订单商品数量、订单备注、订单创建时间。
		   订单子项：id、订单id、商品id、商品名称、商品价格、数量、总价。

6：收藏管理：收藏实体：id、用户、商品、商品、收藏时间。

7：评论管理：评论实体：id、用户、商品、评论类型、评论内容、评论时间。
```

#### 前台功能需求

```
1：首页商品展示
2：购物车：实体：id、商品id、用户id、商品名称、商品图片、商品价格、商品数量、金额、添加时间。
3：个人中心页面
```

### 项目前台介绍

#### 首页

![image-20210323113636314](C:/TyporaImage/TyporaImage2021/image-20210323113636314.png)

```txt
1：点击顶部导航栏“集沃商城”字体、“在线商城”图标、中部导航栏“首页”字体，底部导航栏“首页”字体，均会在当前页面进行刷新；
2：点击顶部导航栏“购物车”字体、中部导航栏“购物车”字体，将跳转到“购物车”页面（如下图）；
3：点击顶部导航栏“已买到的商品”、“个人主页”、“收藏的商品”、中部导航栏“个人中心”，将跳转到“个人中心”页面（如下图）相应的部分；
4：点击“某个商品”将跳转到该商品详情页面（如下图）；
5：光标移动到中部分类栏上，将显示该主分类的子分类，点击“某个子分类”，将筛选出该子分类下的商品，点击商品将跳转到商品详情页面；
6：搜索框可进行简易的搜索功能实现。
```

#### 商品详情页面

![image-20210323115335661](C:/TyporaImage/TyporaImage2021/image-20210323115335661.png)

```
1：商品详情页的顶部导航栏，同首页的顶部导航栏跳转结果一样；
2：商品信息：价格、运费、累计售出、评论、浏览量、购买数量、库存、加入购物车；
3：商品图片展示，底部是图片的导航；
4：热门推荐，并没有用推荐算法进行实现；
5：商品详情展示商品具体信息（未实现）；
5：商品评论：购买后可评论。
```

#### 购物车页面

![image-20210323120547781](C:/TyporaImage/TyporaImage2021/image-20210323120547781.png)



```
1：确认购物清单操作，点击“确认并填写订单”将跳转到步骤二；
2：填写收货地址、备注等信息，点击“确认提交订单”将跳转到步骤三；
3：显示“下单成功”信息，并提示该订单的编号与总价；
```

#### 个人中心页面

![image-20210323121233034](C:/TyporaImage/TyporaImage2021/image-20210323121233034.png)

```
1：个人中心页的顶部导航栏，同首页的顶部导航栏跳转结果一样；
2：个人中心页面导航，点击后将在右边展示具体信息
```

**然后再去理解这个项目中的“前后台分离”，并用自己的话描述出来**

建立一个在线商城前台，供用户在上面购买商品下单，再建立一个商城后台管理系统，供商家在上面处理用户的订单。
